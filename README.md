# MovieVerse AI

An intelligent Agentic AI movie assistant that provides personalized movie recommendations, watch orders, and detailed information using a modern ChatGPT-like interface.

## Features

- 🤖 Agentic AI architecture using LangGraph
- 🎬 Real-time movie data from TMDB
- 💬 Conversational AI powered by Cohere
- 🎯 Intelligent tool selection
- 📚 Conversation memory and context
- 🎞️ Movie recommendations and watch orders
- 🎭 Actor information and filmographies
- 🌙 Dark/Light mode
- 📱 Responsive design
- 🎨 Modern UI with Tailwind CSS

## Tech Stack

### Backend
- FastAPI (Python 3.12+)
- LangGraph
- Cohere API (command-a-03-2025)
- TMDB API
- HTTPX
- Pydantic

### Frontend
- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Shadcn UI
- React Query
